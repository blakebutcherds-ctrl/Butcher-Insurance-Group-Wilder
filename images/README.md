# Wylder / Port St. Lucie photos

Use **official** Wylder and Brystol North at Wylder photos here for the landing page.

## Where to get images

1. **Live Wylder (Port St. Lucie)**  
   [https://livewylder.com/](https://livewylder.com/)  
   Use hero and community shots from the homepage and neighborhood pages (Brystol, Glynlea, Four Seasons, etc.).

2. **Dream Finders – Brystol North at Wylder (41 community photos)**  
   [Brystol North at Wylder – View All 41 Photos](https://dreamfindershomes.com/new-homes/fl/st-lucie/wylder/)  
   Use the community gallery for accurate Brystol/Wylder homes and amenities.

## File names to use in this folder

- **hero.jpg** – Main hero background (e.g. one strong community or home shot).
- **gallery1.jpg**, **gallery2.jpg**, **gallery3.jpg** – Gallery section (e.g. homes, pool, amenities).

After saving these files here, the site will use them when the image URLs in `index.html` point to `images/hero.jpg` and `images/gallery1.jpg`, etc.
